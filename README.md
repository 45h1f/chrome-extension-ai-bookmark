# AI Tools Bookmark Hub (Chrome Extension)

A clean and modern Chrome extension popup that helps you quickly access popular AI tools.

## Features

- AI tools list with:
  - Name
  - Category
  - Icon
- Search by tool name
- Filter by category
- Automatic ranking by usage count (most-used tools on top)
- Quick Notes section with auto-save in localStorage
- Reset usage button

## Project Files

- `manifest.json`
- `popup.html`
- `popup.css`
- `popup.js`

## How to Use

1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder: `ai-bookmark`.
5. Pin the extension and open it from the toolbar.

## Notes

- Usage counts and notes are saved in browser localStorage.
- The list starts with default popularity scores and then learns from your clicks.
